Welcome to my meeting application with collaborative whiteboard and chat.


About the project

Client side

The client side build with Angular.

libraries That I used for client side:

  • Bootstrap for style
  • socket.io-client for real time comunication in whiteboard and chat
  • file-saver for save whiteboard copy

Main feathers

  • Sign up and login with name and password
  • Auto login when refresh and auto sign out when token expired
  • collaborative whiteboard for drawing
  • Chat with public rooms and private room with password

Server side

The server side build with NestJS.
